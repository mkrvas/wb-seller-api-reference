# Supplies FBW — поставки на склады WB

## Назначение

Раздел добавлен автоконвейером: таблица ниже генерируется из официальной
спеки. Рукописные заметки (лимиты, грабли) наполняются по мере боевого
использования — пока считать файл справочным, не боевым.

Формирование и отслеживание поставок FBW на склады WB: опции приёмки, список
поставок и их детали, товары и упаковка (грузоместа) поставки, транзитные
тарифы, список складов. Это **отдельное семейство** от FBS-поставок из
`11-marketplace.md` — другой хост, другая версия API. Штрихкоды товаров для
поставки берутся из Content API (карточки можно создать в песочнице Контента).

**Хост:** `supplies-api.wildberries.ru`  
**Scope токена:** Supplies (управление FBS/FBW поставками)  
**Rate limit:** не задокументирован в справочнике — уточнять в ЛК при выпуске токена, не считать 300 запр/мин Marketplace API применимыми к этому хосту.  
**Версия:** /api/v1/

Хост подтверждён per-path записями `servers:` в спеке 07-orders-fbw; корневого
поля `servers` в спеке нет, поэтому `--list` показывает пустой список. Совпадает
с таблицей `00-overview.md` (`Supplies (Поставки) → supplies-api.wildberries.ru`,
`/api/v1/`).
