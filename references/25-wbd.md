# WBD

## Назначение

Расшифровка аббревиатуры не подтверждена (спеки нет в открытом доступе);
встречались варианты «Wildberries Digital» и «WB Доставка» — уточнится,
если WB опубликует спеку.

Файл — заготовка. Официальный портал `dev.wildberries.ru` **не публикует**
спеку этой категории: запрос `yaml/ru/14-wbd.yaml` возвращает
`{"error":"File not found"}` (проверено зондом 2026-07-23, включая
Playwright-обход анти-бот-защиты). Поэтому категория исключена из ночного
автообновления — иначе политика «всё или ничего» роняла бы весь прогон.

**Хост:** не подтверждён — без спеки узнать нельзя, значения не выдумываю.
**Scope токена / Rate limit / Версия:** не подтверждены по той же причине.

> Если WB опубликует спеку на портале — вернуть закомментированную запись
> `14-wbd` в `scripts/mapping.yaml`, и таблица ниже заполнится ночным
> автообновлением сама.
